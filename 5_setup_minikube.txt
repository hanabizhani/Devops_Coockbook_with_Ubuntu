Wget minikube binary file:
  - sudo wget https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64
  
You can check existance of binary file with ls command 
 
Then, you should copy minikube binary to path:
  - sudo cp minikube-linux-amd64 /usr/local/bin/minikube
  - sudo chmod 755 /usr/local/bin/minikube
  
Check if minikube is in the path:
  - which minikube
  
Now Minikube is in the path and minikube command is recognized:
  - minikube version
